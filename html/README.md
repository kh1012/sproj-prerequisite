# Tools
Seed Project의 기본적인 Tool은 다음과 같습니다.

- Chrome (Browser) - `Edge, Firefox 등을 사용하셔도 무방 합니다.`

다른 Tool을 사용하여도 무방하지만 모든 Description은 위 Tool에 기반 합니다.
